Full-stack platform connecting street vendors with suppliers for price comparison and easy ordering. Built with React, Tailwind CSS, Node.js, Express, and MongoDB.

## Features

- **Price Comparison**: Vendors can compare prices from multiple suppliers.  
- **Order Management**: Place and track orders easily through the platform.  
- **Responsive Design**: Works seamlessly on both mobile and desktop devices.  
- **User-Friendly Interface**: Intuitive and simple navigation for vendors.  
- **Scalable Backend**: Built with Node.js, Express, and MongoDB to support future growth.  

## How It Works

- **Frontend**: React and Tailwind CSS for a smooth and responsive UI.  
- **Backend**: Node.js and Express providing RESTful APIs.  
- **Database**: MongoDB for storing vendor and supplier data.  
- **Integration**: Frontend communicates with backend via REST APIs.
